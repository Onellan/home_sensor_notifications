# Home Sensor Notifications

Custom Home Assistant integration for notifying selected `notify.*` targets when selected door/window sensors open, with repeating reminders and a dedicated sidebar panel.

## Features
- Select monitored binary sensors
- Select notify targets
- Enable/disable notifications
- Configure reminder interval in seconds
- Use one shared message for all sensors or custom messages per sensor
- Dedicated sidebar panel for configuration
- Test notification and forced sound delivery from the sidebar panel
